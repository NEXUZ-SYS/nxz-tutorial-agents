# Respostas da Entrevista — NXZ Go

**Data:** 2026-03-28
**Produto:** NXZ Go
**Entrevistado:** Walter Frey

## 1. Diferenciais do Produto

- Multi-adquirente: funciona com Stone, Cielo, PagBank, SumUp, Pagarme — cliente escolhe
- Offline-first: funciona sem internet e sincroniza quando reconecta — nunca para de vender
- Funcionalidades ricas de cada modo (PDV, Totem, KDS, Smart POS) — cada modo com funcionalidades proprias
- **NAO usar "4 modos em 1 app" como argumento de marketing** — pouco atrativo comercialmente

## 2. Modelo Comercial

- Vendido como parte do ecossistema NXZ (junto com NXZ ERP)
- Nao e vendido separado — faz parte do pacote

## 3. Principal Transformacao

- **Autoatendimento:** reduz filas, libera funcionarios, aumenta ticket medio — cliente faz pedido sozinho
- **Velocidade operacional:** pedido flui automaticamente do caixa/totem pra cozinha sem papel, sem grito

## 4. Destaques Especificos

- **Whitelabel/marca propria:** cliente personaliza visual (cores, logo, screensaver) com sua marca
- **Smart POS:** PDV completo rodando direto na maquininha — sem tablet extra
