# Checkpoint: Revisao do Rascunho

Apresente o tutorial rascunho ao usuario para revisao antes de capturar as screenshots.

## Perguntas

### 1. O tutorial esta adequado?

Mostre o tutorial completo e pergunte:
- O conteudo esta correto e completo?
- Os passos estao claros?
- Falta alguma funcionalidade importante?
- O nivel de detalhe esta adequado para a persona?

### 2. Deseja ajustar algo antes de capturar as telas?

- Aprovar e seguir para captura de screenshots
- Solicitar ajustes no texto
- Adicionar/remover passos
