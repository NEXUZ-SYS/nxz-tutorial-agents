# Checkpoint: Configuracao do Teste

Colete as informacoes necessarias para executar o teste do Captain.

## Perguntas

### 1. Qual time deseja testar?

- Suporte Tecnico
- Implantacao
- Comercial
- Todos os times

### 2. Qual periodo de conversas?

Exemplos:
- Ultimos 7 dias
- Ultimos 30 dias
- Periodo customizado (informar datas)

### 3. Quantas conversas por time?

- 20 conversas (amostra rapida)
- 30 conversas (amostra media)
- 50 conversas (amostra completa)

### 4. Qual o ID do assistant do Captain?

Informe o ID numerico do assistant configurado no Chatwoot (ex: 1, 2, 3).
Se nao souber, o agente tentara descobrir via API.
