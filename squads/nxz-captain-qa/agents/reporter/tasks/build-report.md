output: Relatorio em Markdown estruturado

## Process

1. Escrever resumo executivo
2. Montar tabelas de metricas por time
3. Selecionar 3-5 conversas exemplares (boas e ruins)
4. Listar padroes de falha com exemplos
5. Identificar gaps na knowledge base
6. Escrever recomendacoes priorizadas

## Output Format

Markdown estruturado seguindo o template do relatorio.

## Quality Criteria

- Resumo executivo em 1 paragrafo
- Tabelas com scores corretos
- Minimo 3 recomendacoes acionaveis
- Exemplos concretos de conversas

## Veto Conditions

- Relatorio sem resumo executivo
- Relatorio sem recomendacoes
