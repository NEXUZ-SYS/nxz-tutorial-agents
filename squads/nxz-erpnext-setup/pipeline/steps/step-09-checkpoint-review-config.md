# Step 09: Revisão da Configuração

## Contexto
Caio executou a configuração em camadas. O log mostra o que foi criado,
o que falhou e o que precisa de ação manual.

## Perguntas

### A configuração foi executada corretamente?
Revise o log. Se possível, entre no ERPNext via browser (`http://localhost:<porta>`, login como `Administrator` ou o user dedicado) e verifique alguns itens visualmente.

1. Tudo OK — prosseguir para auditoria automática (Vera)
2. Alguns itens falharam — re-executar os que falharam
3. Problemas graves — pausar e investigar
4. Itens manuais pendentes — vou executar e depois prosseguir
